How to set up your email signature

  1. Gmail: Settings → See all settings → General → Signature → Create new → paste the HTML using the source editor (<>) button in Rich Text mode. Or use "Insert from Drive" if pasting raw HTML.
  2. Outlook (Web): Settings → View all Outlook settings → Mail → Compose and reply → New signature → paste here.
  3. Apple Mail: Mail → Settings → Signatures → create new → drag and drop this HTML file directly into the signature editor.
  4. Google Workspace / Business email: Works the same as Gmail — the SVG logo will render inline without needing an external image host.
The logo is embedded as inline SVG — no external image hosting needed. It will render correctly in all modern email clients including Gmail, Outlook Web, and Apple Mail.
